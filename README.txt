This is a Chess game made using python's pygame module.

Noteable Features: 

Highlights all possible moves and excludes moves that would render the friendly king in check. 
Indicates when king is in check. 
Flips board upon turn end.

Features that should be added:

Pawns need to transform into piece of choice when reaching the end of the board.
Checkmate victory.

Features that could be added:

Main menu.
Load and save game features.
Multiplayer capabilies.
